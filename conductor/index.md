## Project Overview

**MaverickMCP** is a personal-use MCP server for Claude Desktop that provides professional-grade stock analysis, technical indicators, stock screening, and portfolio tracking.

### Key Capabilities

- Real-time and historical market data access with caching
- Advanced technical analysis (RSI, MACD, Bollinger Bands, etc.)
- Multiple stock screening strategies (Maverick Bullish/Bearish, breakouts, supply/demand)
- Portfolio tracking with cost basis, averaging, and live P&L
- Optional “deep research” integrations (OpenRouter + web search providers)
