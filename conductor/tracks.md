## Track Status Legend

- **planning**: Track spec being defined
- **active**: Currently in development
- **paused**: Work temporarily stopped
- **completed**: All phases finished
- **archived**: No longer relevant

## Quick Commands

- `/conductor:new-track` - Create a new track
- `/conductor:status` - View current progress
- `/conductor:implement` - Execute tasks from active track
- `/conductor:manage` - Archive, restore, or delete tracks
